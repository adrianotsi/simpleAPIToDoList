# A Simple API with TS
### This is a simple API to make a ToDo List

- [Node JS](https://nodejs.org/en/) 
- [Fastify](https://fastify.dev/)
- [MongoDB](https://mongodb.com)
- [Prisma](https://www.prisma.io/)

#### To run use:
```
npm install
npm run dev
```
